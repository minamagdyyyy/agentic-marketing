# `marketing-community`

## Overview

Builds community-led growth systems and engagement programs.

## When to use it

Use `/marketing-community` when you need:
- a community strategy
- engagement programming
- moderation structure
- member retention ideas
- advocacy pathways

## Inputs to prepare

- community platform or plan
- audience segments
- participation goals
- moderation capacity
- current member behavior if known

## Deliverables

- community strategy docs
- programming ideas
- operating playbooks
- engagement campaigns

## Output locations

```text
brands/{brand-slug}/campaigns/community/
```

## Related skills

- `marketing-social`
- `marketing-referral`
- `marketing-retention`

## Sample prompts

```text
/marketing-community
Help us design a customer community.
```

```text
/marketing-community
Create a launch plan for a private Slack group for recruiting leaders.
```

```text
/marketing-community
Use our retention goals to design a member engagement program that increases advocacy.
```
