# `marketing-pr`

## Overview

Supports earned media, press outreach, narrative shaping, and reputation-sensitive communications.

## When to use it

Use `/marketing-pr` when you need:
- a press release
- media outreach strategy
- a media list
- thought leadership pitching
- brand reputation support

## Inputs to prepare

- announcement or angle
- spokesperson or source material
- proof points
- target publications or audiences

## Deliverables

- press releases
- media lists
- pitch angles
- outreach plans
- performance summaries

## Output locations

```text
brands/{brand-slug}/campaigns/pr/
```

Often using subfolders such as:
- `press-releases/`
- `media-lists/`
- `performance/`

## Related skills

- `marketing-launch`
- `marketing-influencer`
- `marketing-social`
- `marketing-seo`

## Sample prompts

```text
/marketing-pr
Write a press release for our launch.
```

```text
/marketing-pr
Create a media outreach plan for Northstar AI around recruiting operations automation.
```

```text
/marketing-pr
Use our launch strategy and proof points to build a founder-led thought leadership pitch list.
```
