# `marketing-retention`

## Overview

Focuses on churn reduction, lifecycle recovery, and engagement systems that keep customers.

## When to use it

Use `/marketing-retention` when you need:
- churn diagnosis
- cancel-flow improvements
- win-back campaigns
- dunning sequences
- lifecycle recovery ideas

## Inputs to prepare

- churn or cohort data if available
- lifecycle stage context
- known cancellation reasons
- customer segments
- current retention metrics

## Deliverables

- churn analysis
- retention campaign plans
- win-back flows
- dunning or recovery docs

## Output locations

```text
brands/{brand-slug}/campaigns/retention/
```

## Related skills

- `marketing-email`
- `marketing-analytics`
- `marketing-referral`
- `product-marketing-context`

## Sample prompts

```text
/marketing-retention
Help us reduce churn.
```

```text
/marketing-retention
Build a win-back plan for users who canceled in the last 90 days.
```

```text
/marketing-retention
Use our lifecycle data and customer objections to redesign our cancel flow.
```
